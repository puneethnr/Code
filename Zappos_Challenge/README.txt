To run the application

1. Unzip ZapposIntern.zip
2. Import the unzipped contents as a project into Eclipse.
3. Go to config.properties file and enter a gmail username for the sender.username field and the password for sender.password field.
   This email ID will be used to send E-mails.
4. Run the program as a Java Application. You will be prompted to enter the email ID of the user
   who wants the updates and the product ID/Name of the product about which updates are required.
5. If any relavant products are found, an email will be sent to the email ID entered in step 4.
   The program checks for updates every 5 seconds. So, be sure to stop the program. else be ready to see your Inbox 
   flodded with Zappos deals!!!
